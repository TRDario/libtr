template<class T>
struct tr::ColorCaster< T >

Interface for custom casters for use in color_cast.

To fulfill the requirements for a valid color caster specialization, the following must be present:

static function toBuiltin that takes a value/const reference of type T and returns a builtin color type.

static function fromBuiltin that takes a value/const reference of a BuiltinColor type and returns a value of type T.

If these conditions are properly met, T will fulfill ColorCastableFrom and ColorCastableTo.
