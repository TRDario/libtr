Detailed Description

template<HandleType T, T EmptyValue, HandleDeleter< T > Deleter>
class tr::Handle< T, EmptyValue, Deleter >

libtr examples of usage of this type include various OpenGL and OpenAL objects.

Template Parameters
TUnderlying type used by the handle, must satisfy HandleType.
EmptyValueA value of type T set aside for use as a sentinel of an empty handle.
DeleterThe deleter used by the handle, must satisfy HandleDeleter<T> .
